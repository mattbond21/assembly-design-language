We use round avatars to quickly identify users on the Quizlet platform. No other entity or container should be expressed in a circular shape to avoid confusion between a user and another object type.

Usage

Name Avatar Usage
Small For use in typeahead menus, card meta data, or container meta data
Medium For use in the user profile dropdown menu, or when viewing a list of users in a table
Large For use in profile page headers as the hero
