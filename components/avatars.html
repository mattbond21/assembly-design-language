We use round avatars to quickly identify users on the Quizlet platform. No other entity or container should be expressed in a circular shape to avoid confusion between a user and another object type.

Usage

Name Avatar Example usage
Small - 24px
Typeahead menus, meta data on card containers
Medium - 32px
User profile dropdown menu, or when viewing a list of users in a table (i.e. class members)
Large - 48px
Set page author details
XLarge - 64px
Profile page headers
